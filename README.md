# 💰 Smart Expense Tracker (C++)

## 📌 About the Project
Smart Expense Tracker is a simple console-based application built using C++. It helps users track their daily expenses in an organized and efficient way. The program allows users to add and manage their expenses easily.

I created this project to improve my C++ programming skills and gain practical experience by building a real-world application. This project helped me understand how programming concepts can be used to solve everyday problems.

## ⚙️ Features
- Add new expenses with details (amount, category, etc.)
- View all recorded expenses
- Simple and user-friendly console interface
- Helps in maintaining basic expense records

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (if used)
- File Handling (if used)

## 🚀 How to Run
1. Clone or download the repository
2. Open the project in any C++ IDE (like VS Code)
3. Compile the main.cpp file
4. Run the program
